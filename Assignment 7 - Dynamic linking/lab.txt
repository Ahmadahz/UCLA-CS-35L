3. The following call is a write command to stdout by the loaded stdio library.
write(1, "-0.16055653857469052", 20-0.16055653857469052)    = 20

Summary of commands and their relationships to dynamic linking:
open: opens the libraries with the code to be dynamically linked
read: reads the data about the dynamic library
close: closes the library after dynamic linking
fstat: returns the size of the library, so it can allocate the correct amount of space needed
mmap: creates a new mapping in the virtual address space of the calling process (between library and .text/.data)
mprotect: protects the mapping of the library from any type of attacks
